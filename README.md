## 🔥 Featured Technical Projects

### 🌾 **[Sesame Crop and Weed Object Detection](https://github.com/VanshGit123/Crop-and-Weed-Detection-YOLO)** * **Domain:** Computer Vision, Deep Learning, Edge Computing
* **Overview:** Engineered an end-to-end vision pipeline to automate precision agricultural spraying by accurately localizing weed targets while safeguarding companion crops.
* **Key Achievements:**
  * Achieved an overall **mAP50 of 87.7%** and a targeted **Weed Precision of 90.5%** over 50 epochs using a custom-curated 1,300-image dataset.
  * Optimized model footprint using a **YOLOv8/v11 Nano** architecture to drive inference latency down to **1.6ms per frame** (~625 FPS), meeting real-time execution standards for edge deployment on low-cost field equipment.

### ⚙️ **[Gearbox Fault Predictive Maintenance Pipeline](https://github.com/VanshGit123/Gearbox-Predictive-Maintenance-ML)**
* **Domain:** Supervised Machine Learning, Tabular Time-Series Processing
* **Overview:** Designed an anomalies diagnostic classification engine using raw multivariate data from 4 multi-directional vibration sensors to predict mechanical gear tooth fractures under fluctuating workspace stress loads (0% - 90%).
* **Key Achievements:**
  * Developed a sliding-window data engine to process 400,000 raw time-series lines into clean time-domain statistical descriptors (Mean, Standard Deviation, Peak-to-Peak amplitudes), filtering out ambient simulation background noise.
  * Trained an ensemble **Random Forest Classifier** achieving a balanced **96.0% overall accuracy, precision, and recall**, protecting operations against catastrophic structural failures while dramatically minimizing production downtime.
